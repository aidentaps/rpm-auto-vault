RPM Auto Vault - Admin Dashboard Version

This version is set up for a simple /admin dashboard with these sections:
- Homepage
- Hero image
- Gallery images
- Location
- Contact info
- Availability

To lock down editing:
1. In Netlify, go to Identity and enable it.
2. Set Registration preferences to Invite only.
3. Enable Git Gateway.
4. Invite only you, your dad, and your uncle.
5. Editors then use: https://rpmautovault.com/admin

Why /admin is safe:
- Anyone can visit the page
- Only invited users with login access can edit or publish changes
